A webapp showing current Covid-19 status. Data is fetched from disease.sh, an open API for disease-related statistics. I've also fetched news articles about the corona virus from NewsAPI.

Backend is served on Node js server deployed to Heroku with vanilla js adding functionality to the frontend.
